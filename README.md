# 🛒 Grocery Inventory Management System (Python)

A simple command-line inventory management system built in Python that allows users to manage items through an interactive menu, including adding, editing, deleting, and viewing products.

---

## Overview
This project is a Python-based CLI (Command Line Interface) application that simulates a basic inventory management system. It allows users to manage grocery items efficiently using a menu-driven interface.

The system demonstrates core programming concepts such as input handling, data structures, and function-based design.

---

## Features
- User login system (basic authentication)  
- Display all items  
- Add new items  
- Edit existing items  
- Delete items  
- Interactive menu system  
- Input validation for better user experience  

---

## How It Works
1. The user logs into the system  
2. A menu is displayed with multiple options  
3. The user can:
   - View all items  
   - Add a new item  
   - Edit an existing item  
   - Delete an item  
4. The system updates and displays the inventory accordingly
